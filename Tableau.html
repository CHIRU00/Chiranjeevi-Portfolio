Tableau Visualization Project

Project Description

In this project, I utilized Tableau to create an insightful and interactive dashboard that provides a comprehensive view of various global metrics. The project involved working with different types of visualizations such as maps, bar charts, line charts, and bubble charts. The main highlights include:

  • Map Visualization: Created a world map highlighting different countries based on income levels and average internet usage. This visualization helps in understanding the geographical distribution of income groups and internet penetration.
  • Bar Charts: Designed bar charts to represent country-wise internet usage and birth rates. These charts provide a clear comparison between different countries, making it easier to identify trends and outliers.
  • Bubble Charts: Implemented bubble charts to visualize the relationship between income groups and birth rates. The size and color of the bubbles represent different income groups, offering a quick visual interpretation of the data.
  • Interactive Filters: Integrated filters for country names and income groups to enable users to dynamically interact with the data. These filters enhance the user experience by allowing customized views of the data based on specific criteria.

This project showcases my ability to leverage Tableau's powerful visualization tools to transform raw data into meaningful insights. It also demonstrates my proficiency in creating interactive and user-friendly dashboards for data analysis and decision-making.
